# Liumon 使用指南

## 🎨 现代化交互式界面

### 快速开始

**1. 安装美化库（推荐）**
```bash
pip install rich inquirerpy
```

**2. 启动主程序**
```bash
# 方式 A：双击启动器
liumon.bat

# 方式 B：命令行
python main.py
```

**3. 使用交互式菜单**
- ⬆️⬇️ 上下键选择
- Enter 确认
- 或直接输入数字

---

## 📋 功能菜单

```
🧪 测试核心功能    - 验证系统是否正常
📥 抓取最新数据    - 更新 A 股数据
📊 生成选股报告    - 基于现有数据预测
🚀 运行完整流水线  - 数据 + 预测 (推荐)
📈 运行策略回测    - 历史回测
🎯 训练模型        - 重新训练
📁 查看历史报告    - 浏览所有报告
❌ 退出
```

---

## 🎯 典型使用流程

### 每日操作（5 分钟）

```bash
# 1. 启动程序
python main.py

# 2. 选择 "🚀 运行完整流水线"

# 3. 等待完成（2-5 分钟）

# 4. 查看报告
reports/report_YYYYMMDD.md
```
