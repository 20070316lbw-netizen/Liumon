# Liumon 20 天测试计划

## 📋 测试目标

在正式部署定时任务前，进行 20 天的连续测试，验证：
1. 数据抓取的稳定性

3. 报告生成的可靠性

---

## 🎯 测试方案（针对电脑不一定每天开机）

### 方案选择：手动执行

**为什么选择手动？**
- ✅ 你的电脑不一定每天 16:00 开机
- ✅ Windows 任务计划要求电脑必须开着
- ✅ 手动执行更灵活，想什么时候跑就什么时候跑

**怎么操作？**
```
每天回家后 → 双击 run_manual.bat → 等几分钟 → 看报告
```

## 🔍 测试关注点

### 1. 数据抓取稳定性

**观察指标：**
- 成功率（20 次中成功几次）
- 网络错误频率
- 数据更新延迟

**判断标准：**
- ✅ 成功率 > 80% → 可以接受
- ⚠️ 成功率 < 50% → 需要换数据源

### 2. 模型预测一致性

**观察指标：**
- Top 1 股票是否频繁变化
- 评分是否合理（不会出现极端值）
- 推荐的股票是否可交易

**判断标准：**
- ✅ Top 股票有重复出现 → 模型稳定
- ⚠️ 每天完全不同 → 可能需要调参

### 3. 系统可靠性

**观察指标：**
- 是否出现崩溃
- 报告是否正常生成
- 日志是否有异常

## 🎯 20 天后的决策

### 情况 A：数据抓取成功率高（>80%）

**方案：**
```
1. 继续使用手动执行方式
2. 或者配置定时任务（如果能保证每天 16:00 开机）
3. 或者考虑云服务器（如果办了外币卡）
```

### 情况 B：数据抓取成功率低（<50%）

**方案：**
```
1. 更换数据源（akshare 代替 baostock）
2. 添加重试机制
3. 我帮你修改代码
```

### 情况 C：模型预测不稳定

**方案：**
```
1. 检查是否需要重新训练模型
2. 调整特征权重
3. 优化因子选择
```

## 💡 优化建议

### 如果你经常忘记运行

**创建桌面快捷方式：**
```
1. 右键 run_manual.bat
2. 发送到 → 桌面快捷方式
3. 重命名为 "Liumon 每日选股"
4. 每天看到桌面图标就记得运行
```

### 如果你想自动提醒

**Windows 提醒事项：**
```
1. 打开 Windows 时钟应用
2. 设置每天 20:00 提醒
3. 提醒内容：运行 Liumon 选股
```
